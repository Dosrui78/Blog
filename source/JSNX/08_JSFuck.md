#### 5.8 JSFuck混淆

`JSFuck`（或为了避讳脏话Fuck写作JSF*ck）是一种深奥的JavaScript 程式设计风格。以这种风格写成的代码中仅使用`[`、`]`、`(`、`)`、`!`和 + 六种字元。

|     值      |                        经JSFuck转化后                        |
| :---------: | :----------------------------------------------------------: |
|   `false`   |                            `![]`                             |
|   `true`    |                        `!![]`或`!+[]`                        |
|    `NaN`    |                     `+[![]]`或`+[][[]]`                      |
| `undefined` |                           `[][[]]`                           |
| `Infinity`  | `+(+!+[]+(!+[]+[])[!+[]+!+[]+!+[]]+[+!+[]]+[+[]]+[+[]]+[+[]])` |
